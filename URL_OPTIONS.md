## How to Change URL

### Option 1: Project Rename (Recommended)
Rename the Vercel project to get a cleaner URL:

```bash
# This would give you: phoneway-[random].vercel.app
# Or if available: phoneway.vercel.app
```

### Option 2: Custom Alias (Free)
Add an alias to your existing deployment:

```bash
npx vercel alias set phoneway-rouge.vercel.app phoneway-scale.vercel.app
```

### Option 3: Keep Current URL
**https://phoneway-rouge.vercel.app** works perfectly fine!
